# Site do Eu vi o dorama
## Como executar?

Primeiro, depois que ambiente estiver pronto (NVM, Yarn...), rode o comando

```bash
yarn dev
```

Abra a url [http://localhost:3000](http://localhost:3000) para ver o resultado.
